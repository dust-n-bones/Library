Έκδοση 1.5EΜ
-Απομάκρυνση προτύπων σχεδίασης
-Προσθήκη διαχείρισης δανειζομένων
-Μετάπτωση της διεπαφής χρήστη από Jigloo σε Windows Builder

Έκδοση 1.2JPA
- Λόγω της προσθήκης των Web Services απαιτείται JDK 1.6
- Μετονομασία της κλάσης DataHelper σε Initializer και μεταφορά της μαζί με την 
JpaInitializer στον κώδικα παραγωγής έτσι ώστε να μπορούν να τρέξουν τα παραδείγματα
- Προσθήκη Web Services στο service layer για το δανεισμό και επιστροφή αντιτύπου.
- Προσθήκη ui layer με το GUI για δανεισμό και επιστροφή αντιτύπου με χρήστη του MVP pattern 
Έκδοση 1.1JPA
- Προσθήκη υποστήριξης JPA
- Προσθήκη πεδίων Id στις κλάσεις Author, Book, BorrowerCategory, Loan, Publisher
- Προσθήκη του πακέτου jpadao για την υλοποίηση του DAO για JPA
- Προσθήκη του πακέτου persistence για την αποθήκευση και ανάκληση απλών τύπων
- Αλλαγή του DAOFactory έτσι ώστε να παρακάμπτεται η μοναδικότητα 
  του αντικειμένου ειδικά για έλεγχο
- Αλλαγές στον κώδικα ελέγχου έτσι ώστε οι ίδιοι έλεγχοι να εκτελούνται και με
  χρήση της μνήμης και με χρήση του JPA
